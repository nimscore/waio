# Waio

> **Единая платформа виджетов для вашего Linux-рабочего стола**

---

## ✨ Что такое Waio?

Waio — это **универсальная платформа виджетов** для Wayland-композиторов. Создавайте, делитесь и устанавливайте элементы рабочего стола (панели, часы, мониторинг системы, обои) с помощью одной строки импорта.

Забудьте о хаосе из конфигов. Не нужно настраивать 10 разных программ. Один клиент. Один формат. Безграничные возможности.

---

## 🚀 Возможности

| Функция | Описание |
|---------|----------|
| 🎨 **Slint UI** | Нативный рендеринг без Electron и WebView (минимум ресурсов) |
| 🦀 **Rust Core** | Безопасный доступ к системе (CPU, RAM, Сеть, Аудио) |
| 🌙 **Lua Logic** | Пишите свои источники данных и интеграции на Lua |
| 🔗 **Импорт строкой** | Делитесь настройками через `waio://profile/xyz` |
| 📦 **Пакеты** | Устанавливайте бары + обои + шрифты + конфиг WM в один клик |
| 🛡️ **Песочница** | Система разрешений для безопасности всех виджетов |
